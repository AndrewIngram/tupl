# Active Execution Plans

Put substantial in-progress execution plans here.

Rules:

- Start architectural work with a checked-in active plan.
- Move completed plans to [`../completed`](../completed) in the same branch that lands the work.
- Update the matching durable design docs when the plan is completed.

Mechanical diff-based enforcement for this workflow is not implemented yet and is tracked in [tech debt](../tech-debt-tracker.md).
