# `@tupl/schema`

Schema-building APIs for tupl.

Use this package when you only need to define schemas, build executable schemas, or work with schema types and helpers.

## Install

```bash
pnpm add @tupl/schema
```

## Docs

- Repository: <https://github.com/AndrewIngram/tupl>
- Full runtime API: [`@tupl/core`](https://github.com/AndrewIngram/tupl/tree/main/packages/core)
